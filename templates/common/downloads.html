下载Wine游戏助手

    • Deepin v20

    运行游戏前需要安装支持Vulkan的显卡驱动，点击查看显卡驱动安装教程。

    Wine游戏助手已上架Deepin应用商店，在应用商店内搜索”Wine游戏助手“就能找到，点击安装即可。


    推荐从应用商店安装，如需手动下载， 可点击此处下载 0.5.7.1 版安装包 ，双击即可安装。

    如果遇到依赖关系不满足：wine-staging，请在终端执行以下命令安装依赖：
    sudo apt install wine

    • UOS 20

    UOS用户必读： UOS安全中心（deepin-defender）对本应用的性能有毁灭性影响，战网安装速度慢了10倍！
    UOS个人版可能需要参考该帖子中的方法结束安全中心相关进程，或者直接彻底卸载，
    否则游戏的安装和运行速度都会受到很大影响。

    因为UOS限制了非应用商店应用的安装，所以安装前需要打开开发者模式。
    此外，运行游戏前需要安装支持Vulkan的显卡驱动， 点击查看开发者模式和显卡驱动相关教程。

    点击下载 0.5.7.1 版安装包 ，打开开发者模式后，双击即可安装。

    如果遇到依赖关系不满足：wine-staging，请在终端执行以下命令安装依赖：
    sudo apt install wine

    • Debian
      Ubuntu

    运行游戏前需要安装支持Vulkan的显卡驱动，点击查看显卡驱动安装教程。

    点击下载 0.5.7.1 版安装包 （注意与Deepin/UOS的包不同，安装Deepin/UOS的包后应用图标不会出现）

    如果你的桌面环境支持，双击安装包即可安装。如果不支持，请通过apt命令安装：
    sudo apt install ~/Downloads/net.winegame.client_0.5.7.1_amd64.deb

    其中 ~/Downloads/net.winegame.client_0.5.7.1_amd64.deb 是安装包路径。

    注意：只写文件名是不行的，比如以下是错误例子，会报找不到这样的软件包：
    sudo apt install net.winegame.client_0.5.7.1_amd64.deb
    如果安装包在当前目录，必须加./：
    sudo apt install ./net.winegame.client_0.5.7.1_amd64.deb

    如果遇到依赖关系不满足：wine-staging，请在终端执行以下命令安装依赖：
    sudo apt install wine
